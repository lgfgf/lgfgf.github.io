# 千幻物证库

> 公开、可验证的历史证据数据库 · 持续更新中

本项目旨在整理并公开可交叉验证的历史事件相关证据数据，所有信息均来自公开渠道，供研究、查证与公共讨论使用。

## 📊 数据说明

- **数据文件**：[`evidence_db.csv`](https://raw.githubusercontent.com/lgfgf/lgfgf.github.io/main/evidence_db.csv)
- **字段含义**：
  - `时间`：事件发生时间（格式如 `2024-01-01` 或 `2024年`）
  - `地点`：国家/城市/区域
  - `事件`：简要描述
  - `来源`：原始出处（媒体、报告、档案等）
  - `备注`：补充信息或交叉验证线索

## 🔍 如何使用

1. **在线访问**：[https://lgfgf.github.io](https://lgfgf.github.io)  
   → 查看介绍并下载 CSV 文件
2. **本地分析**：
   - 用 Excel / WPS 打开 `.csv`
   - 用 Python（pandas）、R 等工具进行统计分析
3. **引用建议**：请注明来源及本仓库链接

## ⚠️ 使用声明

- 本库不持立场，仅提供事实性记录。
- 数据真实性依赖原始来源，请自行判断与核实。
- 不对任何误用或误解负责。
